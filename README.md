# Leonardo — Italian Mediterranean Dining App
Artisanal Neumorphic UI Design for Surat's most iconic rooftop restaurant.

## Live Figma Design
[View Full Design on Figma](https://www.figma.com/design/RyM7TM8QxDbPBpP9B1Hks5/leonardo-cafe-app?node-id=0-1&t=N6WAecEJnhkuJyBM-1)

## Overview
Leonardo is a concept mobile + web app UI for **Leonardo Rooftop Restaurant**, an Italian-Mediterranean dining experience in Surat, India. The design system is built around **Artisanal Neumorphism** — soft shadows, warm cream tones, and tactile card surfaces — crafted to reflect the restaurant's refined, old-world Italian identity in a digital format.

Designed in Figma by **Drashti Jiyani** as a UX portfolio project.

---

## Screens

### 🏠 Home — Artisanal Neumorphism
- Hero banner with cinematic rooftop photography
- "Benvenuti" welcome section with founder's note
- Preview menu items with pricing
- Customer testimonial section
- Footer with open hours, location, and contact

### 🍽️ Digital Menu — Neumorphic Style
- Category tabs: Antipasti · Signature Pastas · Wood-Fired Pizza · Mains · Desserts
- Dish cards with photography, description, and pricing
- Chef's recommendation and origin badges
- Responsive layout for both mobile and desktop

### 🖼️ Gallery — Neumorphic Masonry
- Filter tabs: All · Ambiance · Food · Cocktails
- Masonry grid layout for atmospheric restaurant photography
- Closing quote + "Book Your Experience" CTA

### 📅 Reservations — Neumorphic Style
- 3-step reservation flow: Date → Party Size → Time Slot
- Time slot availability indicators (Available / Limited / Sunset Seating)
- Occasion and special requests fields
- Reservation summary panel (desktop)
- Confirm Reservation CTA with cancellation policy note

### 🔲 Bottom Navigation Bar
- Neumorphic floating nav with Menu · Gallery · Reservations
- Active state highlight treatment
